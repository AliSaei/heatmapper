Accepted File Formats

  • Distance matrix or coordinates list upload
    • Tab separated with extensions .txt, .dat, .tsv, or .tab
    • Comma separated with extension .csv
    • First sheet of an Excel workbook with extension .xlsx
  • PDB file upload
    • Protein Data Bank file, as used by the RCSB Protein Data Bank. The atom coordinates from the first chain are used.

Data Structure

  • Distance matrix upload: Coordinates for a square distance matrix should be given. Optionally, labels may be given in the first row.
  • Coordinates list upload: File should consist of n columns, where n ≥ 1, with each row representing a point in n-dimensional space. Optionally, an additional first column containing data labels may be included.

Range

  • Distance matrix upload: Real numbers ≥ 0.
  • Coordinates list upload: Real numbers.
