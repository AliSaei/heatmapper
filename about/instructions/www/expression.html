Accepted File Formats

  • Tab separated with extensions .txt, .dat, .tsv, or .tab
  • First sheet of an Excel workbook with extension .xlsx

Data Structure

The first row is used to assign column names. The data input must contain at least 2 columns and 2 rows with numeric data. If a column labeled "NAME" exists it will be used to assign row names. If a NAME column is not specified rows will be labeled by numbers based on the order they are listed in. Any additional columns containing strings will be ignored.

Range

First row: column names
NAME column: row names
Other Columns: any positive or negative number

Examples of Valid Input

NAME C1 C2
R1 0 0
R2 0 0

col 1 col 2
-0.5 0
0.5 1
