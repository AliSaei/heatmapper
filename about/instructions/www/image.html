Accepted File Formats

    Grid:
  • Tab separated with extensions .txt, .dat, .tsv, or .tab
  • Comma separated with extension .csv
    Image:
  • A JPEG image with extension .jpg or .jpeg
  • A PNG image with extension .png
  • A TIFF image with extension .tif or .tiff

Data Structure

The grid data input must contain 3 columns; X, Y, and Value. The X and Y columns correspond to the X and Y coordinates respectively. Each combination of X and Y coordinates must be in the data set. The Value column provides the value for the set of coordinates. The first row must contain the names for each column; "x", "y", and "value".

Range

X: positive numbers ranging from 1 to the number of grid columns
Y: positive numbers ranging from 1 to the number of grid rows
Value: any positive number

Examples of Valid Input

x y value
1 1 0
1 2 0
2 1 0
2 2 0

value y x
