The expression function within Heatmapper allows the user to view unclustered expression data, such as that from transcriptomic (microarray or RNAseq), proteomic or metabolomicÂ experiments.

Accepted File Formats

  • Comma-Separated Value with extension .csv
  • Tab separated with extensions .txt, .dat, .tsv, or .tab
  • First sheet of an Excel workbook with extension .xls or .xlsx

Data Structure

  • The first (header) row is used to assign column names.
  • If a column labeled "NAME" exists (optional), it will be used to assign row names.
  • Any additional columns containing non-numeric data will be ignored.
  • Data values can be positive or negative numbers.

Range

  • First row: column names
  • NAME column: row names
  • Other Columns: any positive or negative number

Click on the Expression tab on the main header and select the Upload File radio button, followed by the Choose File button. Alternatively, you may choose the 'Example File' radio button. Note: details about the example may be viewed by clicking the Information Icon. The following plot is created.





You may change the colors of your intensities my clicking in the color boxes and selecting a different color.







Also, you may change the hierarchical clustering method and distance measurement method (for computing distance between rows and columns) as shown below. In this example, we choose 'Centroid Linkage' for the clustering method and 'Pearson' for the distance measuremenet matrix. You may also apply the clustering to rows, columns, or both with the 'Appliy Clustering To' option.







A few other options to try in the menu include:

Scale Type - Allows you to select the direction to scale values in, either by rows or columns.

Color Brightness - Contrast may be enhanced by adjusting the brightness of the colors on the plot.

Download - You may download at any time an image of the plot, or a text file in table format.

You may further adjust the plot, such as naming the axes by clicking on the 'Show Advanced Options' tab. Here you may also adjust the color for 'Missing Data', to better distinguish the data without values on the plot. For this exmaple, we do not change this as there are no missing values.







Different result formats are available. For example, click on the 'Row Dendrogram' plot tab, and you will be shown a diagram depicting phylogenetic relationships based on row similarity, and the nodes that data rows fit under. Two other options are 'Column Dendrogram' and 'Table'.
